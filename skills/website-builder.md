# Website Builder Skill
*Deploy modern, fast, production-ready web apps on edge/serverless.*

## Tech Stack
- **Frontend:** React + TypeScript + Tailwind CSS
- **Deployment:** Vercel or Cloudflare Workers
- **Database:** Supabase (PostgreSQL) or serverless alternative
- **DNS:** CloudFlare

## Core Patterns
- Landing page with hero, services grid, pricing, testimonials, CTA
- Service detail pages (in-depth docs, FAQs, case studies)
- Pricing page with tiered options (Good/Better/Best)
- Contact/intake forms
- Admin dashboard for managing service offerings
- Mobile-first responsive design
- Fast load times (<2s FCP)
- SEO-optimized structure

## Design Principles
- Clean, professional, no fluff
- Clear value prop above the fold
- Social proof (testimonials, case studies)
- Multiple CTAs without being pushy
- Dark mode support
- Accessibility first

## Deployment Checklist
- Domain registered and configured
- HTTPS enforced
- Analytics (Vercel or Plausible)
- Email notifications for form submissions
- Automated backups
- CI/CD pipeline for auto-deploy on git push
