## WEBSITE BUILDING STRATEGY

### RULE #1: ONLY BUILD FOR BUSINESSES WITHOUT WEBSITES

**MANDATORY VERIFICATION (Before Building ANYTHING):**

1. **Google Search:** "[Business Name] [City] website"
2. **Check Results:**
   - If website found → STOP IMMEDIATELY
   - If no website → Verify with secondary search
3. **Secondary Verification:**
   - Google: "[Business Name] [City]" (without "website")
   - Check Yelp listing for website link
   - Check Google Business Profile for website
4. **Only if ALL searches show NO website → Proceed**

### Why This Matters
- **Entire strategy = target businesses WITHOUT websites**
- Building for business with website = WASTE OF TIME
- They already paid for their site
- They won't buy ours
- Zero conversion potential
- Damages credibility

### Red Flags (DO NOT BUILD)
