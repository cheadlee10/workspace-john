## Brand Rules

### Always
✓ Use navy + gold color scheme  
✓ Include star logo (★) in all branding  
✓ Maintain professional, confident tone  
✓ Lead with ROI and results  
✓ Use 3-tier pricing (Good/Better/Best)  
✓ Match visual style across all materials  

### Never
✗ Deviate from core colors without approval  
✗ Stretch or distort the logo  
✗ Use corporate jargon or consultant-speak  
✗ Put logo on busy backgrounds  
✗ Mix other accent colors with navy + gold
