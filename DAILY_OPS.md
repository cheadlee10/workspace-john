# Daily Operations Checklist
*Copy this to your daily log. Check off as you go. This keeps you focused.*

---

## Morning (5 min) — What's happening today?
- [ ] Check leads.jsonl for overnight inquiries
- [ ] Check jobs.jsonl for completed/overdue projects
- [ ] Review today's follow-ups needed
- [ ] Any critical blockers?

---

## Mid-Morning (20 min) — Lead Research
**Pick ONE platform per day, rotate daily:**

**Monday:** Reddit r/forhire + r/slavelabour
- [ ] Sort by "New"
- [ ] Find 3-5 matching posts (excel, python, web, automation)
- [ ] Reply to each with template (modify for context)
- [ ] Copy post details to leads.jsonl
- [ ] Set reminder: Follow up on these in 24h if no response

**Tuesday:** Twitter search (#freelance #hireme #hiring)
- [ ] Find 3-5 recent tweets about hiring/work
- [ ] Reply to each with pitch (keep under 280 chars)
- [ ] Tag relevant accounts for engagement
- [ ] Log successful replies (potential leads)

**Wednesday:** LinkedIn cold outreach
- [ ] Search 3-5 target profiles (business owner / ops manager / CTO)
- [ ] Send personalized connection request + message
- [ ] Wait for acceptance
- [ ] Log to leads.jsonl with profile link

**Thursday:** Reddit + follow-ups
- [ ] Scan Reddit again for new posts
- [ ] Follow up on yesterday's leads (T+24h)
- [ ] Any replies? Move to "proposal_sent" status

**Friday:** Twitter + LinkedIn follow-ups
- [ ] Retweet/engage with previous connections
- [ ] Follow up on LinkedIn messages (if accepted yesterday)
- [ ] Any warm connections? Move to "contacted" status

**Saturday:** Cleanup + bulk follow-up
- [ ] Scan all leads in "new" or "contacted" status
- [ ] Follow up on anything >48h old
- [ ] Mark as "lost" if no response after 2 follow-ups
- [ ] Analyze what worked this week

**Sunday:** REST (or optional: case study building)

---

## Mid-Day (10 min) — Respond to Inquiries
- [ ] Any new leads from yesterday's outreach?
- [ ] Respond to ALL inquiries within 2 hours (use templates)
- [ ] If inquiry needs scope discussion, suggest 15-min call
- [ ] Log response in leads.jsonl as "contacted"

---

## Afternoon (10 min) — Follow-Up Management
- [ ] Any leads waiting on proposal? (status: "proposal_sent")
- [ ] Any proposals >48h old with no response? Send follow-up
- [ ] Any negotiations active? Check Discord for DMs
- [ ] Any wins? Celebrate + add to jobs.jsonl

---

## Late Day (5 min) — Log & Update
- [ ] Update leads.jsonl with all new activity
- [ ] Update jobs.jsonl if deal closed
- [ ] Log observations to memory/observations.md (what worked/didn't)
- [ ] Update KPI tracker in memory/YYYY-MM-DD.md

---

## Important Reminders
- ⏰ **Respond FAST** (<2 hours) = 2x conversion
- 💰 **Offer 3 tiers** = higher average deal size
- 📝 **Follow up** = 50% of deals come from follow-up
- 📊 **Log everything** = data drives decisions
- 🎯 **Stay focused** = one task at a time, no multitask
- 💪 **Celebrate wins** = every deal is a proof point

---

## Daily Goal
**15 minutes of lead research + 10 minutes of follow-up = 1 new lead per day**
**1 lead/day × 5 days = 5 leads/week**
**5 leads/week × 3 weeks = 15 leads/month**
**15 leads × 15% close rate = 2-3 deals/month**
**2-3 deals × $300-500 average = $600-1500/month**

Simple math. Just execute.
