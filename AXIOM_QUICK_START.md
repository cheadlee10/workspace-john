## STEP 3: First Automation - Fiverr Gig Posting (10 minutes)

**Axiom officially supports Fiverr gig creation:**

1. Open Fiverr.com and login to your seller account
2. Navigate to: Fiverr.com → Selling → Gigs → Create a New Gig
3. Click Axiom icon → "New Automation"
4. Name it: "Post Fiverr Gig"
5. Click "Start Recording"
6. Fill out ONE complete gig manually (Axiom records every action)
7. Click "Stop Recording" when done
8. Save automation

**Now you have a bot that can post gigs automatically.**

---

## STEP 4: Set Up Order Monitoring (5 minutes)

1. Navigate to Fiverr.com → Selling → Orders
2. Click Axiom icon → "New Automation"
3. Name it: "Check New Orders"
4. Record: Check for new orders → Extract order details → Copy to clipboard
5. Set to run every 15 minutes (Settings → Schedule)
6. Enable "Headless mode" (runs in background)

**Now Axiom checks orders 24/7 without you watching.**

---

## STEP 5: Test It (5 minutes)

1. Run "Post Fiverr Gig" bot once
2. Verify gig appears on Fiverr
3. Run "Check New Orders" bot once
4. Verify it finds your existing orders

**If both work → SUCCESS → You're automated**

## STEP 6: Enable Headless + Scheduling (3 minutes)

**For each automation:**
1. Click kebab menu (⋮) → Settings

3. Enable "Schedule" → Set frequency (every 15 min for orders)
4. Click "Save"

**Now bots run automatically 24/7 in background.**
