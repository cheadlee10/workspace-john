# Lee's General Landscaping Website

## Modern, Professional Landing Page

Built with Next.js 14, TypeScript, Tailwind CSS, and Framer Motion.

### Features

✅ **Hero Section** - Full-screen with parallax effect, compelling headline, dual CTAs
✅ **Social Proof Bar** - 30 years, 5,000+ properties, 600+ reviews, 100% satisfaction
✅ **Services Grid** - 6 services with icons, descriptions, and feature lists
✅ **Scroll Animations** - Elements fade in as you scroll (smooth, professional)
✅ **Testimonials** - 3 real customer reviews with ratings
✅ **About Section** - 30-year story, trust badges, team photo
✅ **Service Areas** - Greater Seattle coverage map
✅ **Mobile Responsive** - Perfect on all devices
✅ **SEO Optimized** - Meta tags, OG tags, semantic HTML
✅ **Performance** - Optimized images, lazy loading, fast load times

### Quick Start

```bash
cd lees-landscaping
npm install
npm run dev
```

Open http://localhost:3000

### Deploy to Vercel

1. Push to GitHub
2. Connect to Vercel
3. Deploy (automatic)

### Design Principles Used

- **Color Palette**: Green (primary) + Earth tones (natural, professional)
- **Typography**: Outfit (headlines) + Inter (body) - modern, readable
- **Layout**: F-pattern, generous white space, clear hierarchy
- **Animations**: Subtle scroll reveals, parallax hero, smooth transitions
- **Psychology**: Social proof above fold, clear CTAs, trust signals throughout

### What Makes This Better Than Current Site

1. **Modern hero** with full-screen image (vs plain header)
2. **Scroll animations** make site feel alive (vs static)
3. **Visual hierarchy** guides user to action (vs text-heavy)
4. **Social proof bar** immediately shows credibility (vs buried stats)
5. **Service cards** with icons and features (vs plain list)
6. **Testimonials** with real names and locations (vs generic quotes)
7. **Mobile-first** responsive design (vs desktop-only)
8. **Performance** optimized for speed (vs slow load times)
9. **SEO** structured for search engines (vs missing meta tags)
10. **Conversion-focused** CTAs throughout (vs single contact form)

### Conversion Optimization

- Hero CTA: "Call for Free Estimate" (action-focused)
- Secondary CTA: "View Our Services" (discovery path)
- Social proof: Stats bar immediately after hero
- Multiple CTAs: Every 2-3 sections (maintain momentum)
- Trust signals: Licensed, bonded, insured badges
- Phone number: Prominent in nav and hero (low friction)
- Reviews: 600+ five-star reviews (credibility)
- Service areas: Clear coverage map (qualify leads)
