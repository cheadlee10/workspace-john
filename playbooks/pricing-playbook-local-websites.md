# Pricing Playbook — Local Website Offers (NorthStar)

## Pricing Philosophy
- Lead with value (new jobs, calls, bookings), not pages/features.
- Offer 3 tiers every time (Good/Better/Best).
- Use **scope-down**, not rate-down, on negotiations.
- Anchor on Tier 2 as default recommendation.

---

## Standard Package Card
## Good — $500 (3–5 days)
- 5 core pages
- Mobile optimization
- Contact form + map
- Basic local SEO setup

## Better — $1,000 (5–7 days) **[Default Anchor]**
- Everything in Good
- Gallery + testimonials
- Lead capture improvements
- Basic booking/request workflow
- Analytics setup

## Best — $2,000 (7–10 days)
- Everything in Better
- Payments/advanced booking automation
- Reporting dashboard basics
- Priority implementation + support

## Maintenance Retainers
- Lite: $100/mo (minor edits + uptime checks)
- Growth: $200/mo (content updates + optimization)
- Priority: $300/mo (fast-turn changes + monthly reporting)

---

## Pricing Rules
1. Quote all three tiers in every proposal.
2. If urgent timeline (<72h), add 20–30% rush fee.
3. If unclear scope/multiple approvers, add risk buffer 15–20%.
4. If they ask cheaper, remove features or phase delivery.
5. Never discount below profitable floor unless strategic (explicit approval).

---

## Discovery-to-Price Mapping
- **Low complexity + low urgency:** Good
- **Most SMBs (recommended):** Better
- **Needs automation/payments/reporting:** Best

### Add-On Menu
- Additional page: +$100 each
- Copywriting assist: +$150
- Booking integration custom logic: +$250
- Stripe/payment flow setup: +$300
- Extra revision round: +$100

---

## Negotiation Script Snippets
- "To keep quality intact, I can lower scope instead of cutting core build quality."
- "If budget is the blocker, we can start on Good and upgrade after launch."
- "Most clients choose Better because it balances speed and ROI."

---

- [ ] Scope defined in writing
- [ ] Revision limit defined

- [ ] Asset dependencies clear
- [ ] Maintenance offer presented at close
