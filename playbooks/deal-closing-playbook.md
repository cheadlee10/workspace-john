# Deal Closing Playbook — From Inquiry to Payment

## Phase 1: Initial Contact (T+0)
**Goal:** Acknowledge inquiry + show you understand their problem

### Action Items
- [ ] Respond within 2 hours (speed kills in freelancing)
- [ ] Personalize response (show you read their post/email)
- [ ] Restate problem in 1 sentence
- [ ] Offer 3 tiers if scope is unclear
- [ ] Add one credential/proof point ("Built similar for X clients")
- [ ] Include clear CTA ("Ready to discuss?")

### Template
"Hi [Name], I can handle this. Built similar for [X clients] in [Y industry]. Quick scope:
- [Solution A]
- [Solution B]
- [Solution C]

Timeline: [X days]. Investment: $[range]. Ready to discuss? Let me know."

### Key Metrics
- Response time: <2 hours
- Personalization: Must mention their specific project
- Credibility: Must include 1 past example
- CTA: Must be clear and specific

---

## Phase 2: Qualification (T+2h to T+24h)
**Goal:** Understand budget, timeline, scope depth

### Ask These Questions
1. "What's your timeline?" (ASAP = higher price)
2. "What's your budget range?" (if unclear)
3. "Have you tried other solutions?" (context for scope)
4. "Who else is involved in this decision?" (decision-maker check)
5. "When would you want to start?" (commitment signal)

### Red Flags → Increase Price
- Vague scope ("make it work")
- Unrealistic timeline (30 hours in 3 days)
- No budget ("let's see what it costs")
- Multiple decision-makers + back-and-forth expected
- Client seems difficult/demanding

→ Add 20-30% to price if any red flags.

### Green Lights → Move Fast
- Clear scope
- Budget available
- Decision-maker present
- Wants to start ASAP
- Professional communication

→ Send proposal same day.

---

## Phase 3: Proposal (T+24h to T+48h)
**Goal:** Convert qualified lead to paying customer

### Proposal Elements
✓ **Project summary** (1 paragraph, their words)
✓ **Scope breakdown** (what you'll deliver)
✓ **Timeline** (specific dates)
✓ **3 pricing tiers** (Good/Better/Best)
✓ **Payment terms** (net-15 or upfront?)
✓ **Next steps** (what happens after they pick)
✓ **Portfolio attachment** (1 similar past project)

### Proposal Length
- Keep under 1 page for <$500 projects
- 1-2 pages for $500-$2000
- 2-3 pages for $2000+

### Pricing Tiers
Always Good/Better/Best:
- **Good:** Base scope, timeline [X days], price $[P1]
- **Better:** Base + 1 upgrade, timeline [X days], price $[P2] (usually 2x)
- **Best:** Full service, timeline [X days], price $[P3] (usually 3-4x)

Client picks tier → higher-tier adoption usually 30-40%.

### Example Tiers
**Excel Audit:**
- Good ($150): Formula review + 5 optimizations
- Better ($300): Audit + optimization + basic macro
- Best ($500): Full audit + advanced automation + training

**Python Script:**
- Good ($99): Simple script, 1 use case
- Better ($299): Advanced features + error handling
- Best ($999): Full production system + monitoring

## Phase 4: Negotiation (T+48h to T+72h)
**Goal:** Close the deal, handle objections

### Common Objections & Responses

**"It's too expensive."**
→ "I can do a simpler version for [lower price], or we can discuss which features matter most. What would work for you?"
(Don't just cut price — reduce scope instead)

**"Can you do it faster?"**
→ "I can rush it to [X days] for an extra [X%], or stick to [original timeline]. What's the deadline?"

**"Can I see examples?"**
→ Send 1-2 portfolio pieces + 1 client quote
(Don't send 10 — too much noise. Be selective.)

**"Why is it so much?"**
→ "This includes [specific: testing, documentation, training, unlimited revisions, etc]. At $[rate/hr], it's [comparison: less than agency, faster than freelancer, etc]."

### When to Walk Away
❌ Client negotiates below 20% of original quote
❌ Scope keeps expanding (scope creep)
❌ Budget suddenly shrinks
❌ Decision-maker changes partway through
❌ Client is disrespectful or demanding

→ Say: "Thanks for considering me. Let's circle back if budget/timeline changes. Good luck!"

### Closing Signals
✓ Client asks about payment options
✓ Client asks about start date
✓ Client asks about revisions
✓ Client asks about references

→ When you see any of these: "Ready to move forward? I can start [date] upon payment."

---

## Phase 5: Close & Payment (T+72h to T+5 days)
**Goal:** Get payment, start delivery

### Closing Line
"I'm excited to get started. Here's how we'll proceed:

1. You select a tier (Good/Better/Best)
2. I'll send you a payment link
3. Upon payment confirmation, I'll start immediately
4. You'll get [X updates/deliverables] along the way
5. Final delivery by [date]

Which tier works best for you?"

### Payment Collection
✓ Stripe payment link (for most projects)
✓ Invoice with net-15 terms (for retainers)
✓ 50% upfront / 50% on delivery (for high-risk projects)

→ Log all payments to `jobs.jsonl` immediately.

### Kickoff Checklist
- [ ] Confirm project details in writing
- [ ] Send deliverables checklist
- [ ] Schedule kickoff call (15 min)
- [ ] Ask for any assets/access needed
- [ ] Set weekly update cadence

---

## Phase 6: Delivery & Upsell (T+5 days to completion)
**Goal:** Over-deliver, set up for repeat business

### Delivery Best Practices
✓ **Weekly updates** (even if just "on track")
✓ **Early delivery** (beat the deadline)
✓ **Document everything** (formulas, code, process)
✓ **Video walkthrough** (3-5 min showing how to use)
✓ **Offer training** (free 30-min call to learn the system)
✓ **Ask for feedback** (testimonial / case study permission)

### Upsell Opportunities
After delivery, pitch:
- Ongoing retainer ("I can maintain this for $X/month")
- Add-ons ("Want me to add X feature?")
- Related service ("While I'm here, I could also...")
- Referrals ("Know anyone who needs this?")

### Follow-up Sequence
**T+0 (delivery):** Send deliverables + walkthrough video
**T+3 (post-delivery):** "How's it going? Any questions?"
**T+7 (check-in):** "Want to set up that training call?"
**T+14 (retainer pitch):** "I can maintain/optimize this for $X/month. Interested?"
**T+30 (testimonial ask):** "Would you mind writing a quick testimonial?"

## Weekly Review
Every Friday:
1. How many new leads did I generate?
2. How many proposals did I send?
3. How many deals did I close?
4. What was my conversion rate?
5. What went wrong? How do I fix it?

Update `memory/observations.md` with findings.

---

## Anti-Patterns (Don't Do This)
❌ Spend 2 hours on a proposal for a $50 project
❌ Say yes to every scope request (manage creep)
❌ Forget to follow up (50% of deals come from follow-up)
❌ Cut price instead of reducing scope
❌ Work before getting paid (upfront or net-15 only)
❌ Skip documentation (costs you repeat business)
❌ Stop communicating once project starts
❌ Deliver mediocre work thinking it's "good enough"
