## 4) Simple Demo Flow Script (3–4 Minutes)

### Demo Objective
Show one realistic workflow transforming from manual chaos to reliable automation.

### Scenario
Inbound lead form submission triggers CRM creation, owner assignment, team alert, follow-up task, and dashboard update.

---

### Talk Track (time-boxed)

**0:00–0:30 — Problem Setup**  
“Most teams don’t have a software shortage—they have a handoff gap. Leads come in, then manual steps create delays and mistakes.”

**0:30–1:00 — Before Snapshot**  
Show inbox + spreadsheet + manual notes.  
“This is the old state: copy-paste work, unclear ownership, and inconsistent follow-up.”

**1:00–1:30 — Trigger the Workflow**  
Submit sample lead form.  
“Once this is submitted, the workflow starts immediately.”

**1:30–2:45 — Automation Sequence**  
Walk through each action in order:
1. CRM record created with clean standardized fields
2. Lead owner assigned by territory/service rule
3. Slack/email alert sent to assigned rep
4. Follow-up task created with due date and priority
5. Dashboard KPI updates in real time

Narration line:  
“No one has to remember the next step—the system handles it.”

**2:45–3:20 — ROI Snapshot**  
“On a weekly basis, this removes repetitive admin time and improves follow-up consistency. That means faster response and more pipeline reliability.”

**3:20–3:40 — Close / CTA**  
“We start with a free automation audit, prioritize one high-impact workflow, and deliver quickly.”
