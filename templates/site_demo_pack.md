## 4) Simple Demo Flow Script (For Sales Calls / Loom / Live Walkthrough)

## Demo Goal
Show how NorthStar turns a manual lead-to-client process into an automated, trackable system in under 10 minutes.

### Demo Stack (example)
- Intake form (website form or Typeform)
- Automation layer (Zapier/Make/n8n)
- CRM (HubSpot/Pipedrive/GoHighLevel)
- Email/SMS follow-up
- Dashboard (Google Sheets/Looker Studio/Airtable)

### Step-by-Step Script

**Step 1 — Set the baseline (30 sec)**  
“Right now, most teams manually check submissions, copy data into a CRM, and follow up later. That delay costs deals.”

**Step 2 — Trigger event (60 sec)**  
Submit a sample lead form live.  
Narration: “The moment this form is submitted, automation takes over.”

**Step 3 — Show routing + enrichment (90 sec)**  
Display workflow run:
- Lead is normalized/validated
- Duplicate check is performed
- Record is created/updated in CRM
- Lead is tagged by service type / urgency

Narration: “No copy-paste, no dropped leads, no inconsistent entries.”

**Step 4 — Show follow-up automation (90 sec)**  
Reveal instant personalized email/SMS + internal Slack/Discord alert.

Narration: “Response goes out immediately, while your team receives context to close faster.”

**Step 5 — Show pipeline visibility (90 sec)**  
Open dashboard with key metrics:
- New leads today
- Response-time SLA
- Booked calls
- Conversion by source

Narration: “Leadership gets real-time visibility without waiting on manual reporting.”

**Step 6 — Show exception handling (60 sec)**  
Trigger an edge case (missing phone/duplicate lead).  
Narration: “When data is incomplete, automation flags it and routes it to the right person with instructions.”

**Step 7 — ROI close (60 sec)**  
“Even one automation like this can save [X] hours/week and recover leads that were previously missed. We usually start with one high-impact workflow, prove ROI, then expand.”

### Demo Close CTA
**“Want this mapped to your exact process? We’ll do a 20-minute automation audit and show a custom blueprint.”**
