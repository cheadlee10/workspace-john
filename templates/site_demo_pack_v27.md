## 4) Simple Demo Flow Script (3–5 Minutes)

Use this script for live calls or recorded walkthroughs.

### Demo Goal
Show how one manual process is transformed into an automated flow that saves time and improves reliability.

### Setup (30 sec)
“Today I’ll show a quick before-and-after of a common workflow: incoming data, cleanup, and reporting output. This is the same framework we apply to lead routing, CRM updates, and recurring admin tasks.”

### Step 1 — The Manual Baseline (45 sec)
- Show a raw input source (spreadsheet/form export/email attachment).  
- Narration: “Normally this requires manual copy/paste, format fixes, and repetitive checks.”  
- Point out 2–3 friction points (time, errors, delays).

### Step 2 — Triggered Automation (60 sec)
- Trigger the flow (new row, uploaded file, submitted form).  
- Narration: “Once triggered, the workflow validates fields, standardizes data, and routes outputs automatically.”  
- Show logs/status indicator if available.

### Step 3 — Output & Decision Layer (60 sec)
- Show cleaned dataset + final destination (dashboard, CRM, notification, report).  
- Narration: “Your team now gets consistent outputs without manual prep, so decisions happen faster.”

### Step 4 — Exception Handling (30 sec)
- Show one bad input case.  
- Narration: “If data fails validation, it’s flagged and routed for quick review instead of breaking the whole process.”

### Step 5 — Business Impact Wrap (30 sec)
“Net effect: less repetitive work, fewer avoidable errors, and more time for revenue-driving activity. We then document everything so your team can run it confidently.”

### Demo CTA (10 sec)
“If you want, we can map one of your real workflows and give you a practical automation plan with timeline and expected savings.”
