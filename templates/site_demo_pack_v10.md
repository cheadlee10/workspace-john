## 4) Simple Demo Flow Script (5–7 Minutes)

> Goal: walk a prospect from pain → automation → measurable outcome.

### Demo Structure Overview
1. Identify current bottleneck (1 min)
2. Show “before” process friction (1 min)
3. Run live/visualized automated flow (2–3 min)
4. Show output dashboard + follow-up trigger (1 min)
5. Confirm expected ROI + next step (1 min)

---

### Talk Track (Presenter Script)

**Step 1 — Set context (30–60 sec)**
“Most teams we work with are losing time in the same places: manual follow-up, fragmented data, and repetitive admin. We’ll show how one automation flow fixes all three.”

**Step 2 — Before state (45 sec)**
“Here’s the current process: a lead submits a form, someone manually copies data into a sheet/CRM, another person sends follow-up later, and reporting updates at end-of-week. This creates delays and dropped opportunities.”

**Step 3 — Trigger event (30 sec)**
“Now we’ll submit the same lead form. This is the trigger.”

**Step 4 — Automated sequence (60–90 sec)**
“Immediately, the system creates/updates the CRM contact, tags the lead, sends a personalized SMS/email, creates an internal task, and posts a team alert. No manual handoff.”

**Step 5 — Visibility layer (45 sec)**
“Now in the dashboard, you can see this lead appear in real time—source, stage, response status, and owner. Leadership gets clean reporting automatically.”

**Step 6 — Business result framing (45 sec)**
“This is where clients usually recover 5–20+ hours per week, reduce response lag, and improve consistency in follow-up. The value compounds every week.”

**Step 7 — Close (30 sec)**
“If this matches your bottlenecks, next step is a focused automation audit. We map your highest-ROI workflow and deliver a phased implementation plan.”
