# Local Service Website Blueprint Package

This package pairs with:
- `playbooks/enterprise-local-service-website-build-standard.md`

## Files
1. `local-service-design-system.tokens.json` — design token baseline
2. `local-service-page-blueprint.md` — conversion-focused page section blueprint
3. `local-service-qa-gates-checklist.md` — ship/no-ship quality gates
4. `local-service-delivery-artifacts-checklist.md` — handoff and closure artifacts

## Intended Use
- Start each new local-service web project with these files.
- Copy and adapt for vertical-specific variants (roofing, landscaping, HVAC, etc.).
- Keep versioning in Git to track standard improvements.
