## 6) Demo Flow Script (3-Minute Walkthrough)

Use this script for a live sales call, Loom video, or embedded product demo.

### Demo Goal
Show a before/after workflow transformation with measurable time savings.

### Setup (What to show on screen)
- Tab A: “Before” spreadsheet/inbox/manual workflow
- Tab B: Automation trigger (form submit / new row / scheduled run)
- Tab C: “After” output dashboard + notification/log

### Script
**0:00–0:20 — Context**  
“Here’s the exact process most teams run manually today: data comes in here, someone cleans it up, then someone else sends updates. It works, but it burns hours and creates avoidable mistakes.”

**0:20–0:55 — Pain Snapshot**  
“On a normal week, this takes [X] hours and usually creates [Y] delays—especially around handoffs and follow-ups.”

**0:55–1:40 — Trigger the Automation**  
“I’ll run the same process through NorthStar’s automation flow. One trigger starts intake, cleanup, routing, and status updates automatically.”

**1:40–2:15 — Show the Result**  
“Now you can see the cleaned dataset, assigned follow-up owner, and updated dashboard—without manual re-entry.”

**2:15–2:40 — Quantify Impact**  
“This removes about [X] hours/week, reduces manual errors, and gives leadership real-time visibility.”

**2:40–3:00 — Call to Action**  
“If you want, we can run this same audit on one of your workflows and show the ROI before you commit to a build.”

- **Run My Free Workflow Audit**
- **Show Me My Automation ROI**
