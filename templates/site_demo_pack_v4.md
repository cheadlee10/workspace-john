## 4) Simple Demo Flow Script (3 Minutes)

### Demo Goal
Show one common manual process transformed into a reliable, automated flow.

### Demo Assets (prepare before call)
- `before_raw_leads.csv` (duplicates, missing data, inconsistent fields)
- `after_pipeline_view` (clean records, owner assignment, status tracking)
- `dashboard_summary` (response speed, throughput, conversion)

### Presenter Script (Timestamped)
**0:00–0:20 — Set Context**  
“Most teams aren’t short on tools—they’re short on connected workflows.”

**0:20–0:50 — Show Current Pain**  
“This is the manual intake: duplicate records, missing owners, inconsistent follow-up.”

**0:50–1:30 — Trigger Automation**  
“Now I run one automation. It standardizes data, removes duplicates, assigns owners, and creates follow-up tasks.”

**1:30–2:20 — Show New State**  
“Here’s the cleaned pipeline and updated dashboard. Every lead has an owner, stage, and next action.”

**2:20–2:45 — Tie to ROI**  
“This removes repetitive admin work, speeds up response, and gives leadership live visibility.”

**2:45–3:00 — Close CTA**  
“We can map and launch your first workflow in days. Let’s run a free workflow audit.”

### Quick Objection Responses
- **“We already have a CRM.”** Great—this improves process around your CRM so it gets used consistently.
- **“This sounds complex.”** We start with one workflow and keep scope tight.
- **“What about budget?”** We prioritize quick wins that pay for phase two.
