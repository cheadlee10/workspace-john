## 4) Simple Demo Flow Script (3–5 Minutes)

### Demo Objective
Show one messy manual workflow transformed into a clean, automated system with visible business impact.

### Pre-Demo Setup
- File 1: `before_raw_leads.csv` (duplicates, missing fields, inconsistent formatting)
- File 2: `after_pipeline_view` (clean records, owner assignments, status tracking)
- File 3: `dashboard_summary` (response time, throughput, conversion indicators)
- Optional: on-screen timer for impact

### Live Demo Sequence
1. **Set the frame (30 sec)**  
   “This is the current manual flow: collect data, clean it, assign owners, follow up, then report.”
2. **Show the pain (30 sec)**  
   Highlight duplicate entries, missing owner fields, and stale follow-up timestamps.
3. **Run automation (45 sec)**  
   Trigger one button/script and narrate: “Data is normalized, deduped, assigned, and pushed to follow-up.”
4. **Reveal outputs (60 sec)**  
   Show clean pipeline, owner routing, and updated dashboard metrics.
5. **Tie to outcomes (45 sec)**  
   “This removes ~X manual steps and saves ~Y hours/week while improving response speed.”
6. **Close with CTA (30 sec)**  
   “We can deploy this for your workflow in days. Want a free audit of your current process?”

### Optional Word-for-Word Talk Track
“Most teams don’t need more software—they need their existing tools to work as one system. Here’s the raw intake: duplicate records, missing fields, and inconsistent formatting. I run one automation, and now records are standardized, owners are assigned, follow-up tasks are generated, and leadership has a live dashboard. Same team, same tools, better operating system.”

### Objection Handling (Quick Responses)
- **“We already use a CRM.”** Perfect—this improves the workflow around it.
- **“This sounds complicated.”** We start with one high-ROI process and keep implementation simple.
- **“We need to watch budget.”** We scope quick wins first so savings can fund phase two.
