# V3 Local Service WOW+ Variant Package

Enterprise-grade local-service template iteration focused on stronger premium visuals, faster lead capture, and mobile-first conversion reliability.

## What's New vs V2
- Elevated visual system: richer gradients, glass layering, proof-first hero composition
- Conversion flow upgrade: dual-stage form (quick callback + full quote) with persistent micro-CTAs
- Mobile polish: better thumb ergonomics, snap cards, reduced scroll friction, safe-area sticky action rail
- Trust density upgrade: stronger guarantee language, SLA framing, and proof modules

## Package Contents
- `variant-spec.md` — choreography, behavior rules, and conversion architecture
- `implementation/tokens.v3.json` — updated design and motion tokens
- `implementation/content-model.v3.json` — expanded data model for proof and process storytelling
- `implementation/index.v3.html` — production-ready responsive starter page
- `implementation/qa-wow-v3-checklist.md` — QA gate set for performance, conversion, and mobile UX

## Quick Start
1. Duplicate `implementation/index.v3.html` into the client project.
2. Replace business profile, service area, and offer copy.
3. Apply niche colors from `tokens.v3.json`.
4. Map CMS/API payloads to `content-model.v3.json` keys.
5. Run `qa-wow-v3-checklist.md` before launch.
