# V3 Variant Spec — Local Service WOW+ (Enterprise Iteration)

## 1) Positioning
**Theme:** "Category-leading local pro" — premium execution, predictable communication, low-friction booking.

- Visual tone: cinematic gradient backdrop + bright action accents + soft-glass UI
- Message tone: responsiveness, certainty, and local credibility
- Behavioral tone: decision support first, hard-sell never

## 2) Page Choreography (Single-Service Home)
1. **Utility + Sticky Header**
   - Service area, response SLA, trust score
   - Tap-to-call + quote CTA always accessible

2. **Hero Conversion Cluster**
   - Outcome-led headline
   - 3 proof chips + guarantee microcopy
   - Stage-1 callback form (name + phone)

3. **Proof Bar**
   - Rating, completed jobs, response time, warranty term

4. **Service Outcomes (snap cards on mobile)**
   - Problem -> Result -> ETA microcards

5. **Before/After + KPI Delta**
   - Visual compare panel with quantifiable impact

6. **Process Confidence Rail**
   - 4-step transparent workflow + communication checkpoints

7. **Offer + Risk Reversal**
   - Deadline/seasonal driver + concrete guarantee

8. **FAQ + Objection Control**
   - Price, scheduling, scope, payment, warranty clarity

9. **Final Conversion Hub**
   - Stage-2 full form + click-to-call + service area + trust/legal notes

## 3) Conversion Architecture
- **Primary:** full quote form submit
- **Secondary:** click-to-call
- **Assist:** quick callback capture (above fold + midpage)

### Flow Rules
- Hero = stage-1 capture only (fast intent commit)
- Mid-page = CTA every 1–2 modules
- Final hub = stage-2 detailed capture with optional project notes

### Trust Injection Rules
- Header trust strip
- Hero proof chips
- Offer guarantee block
- Footer credentials + consistency checks

## 4) Mobile Polish Rules (Non-Negotiable)
- Sticky bottom action rail (safe-area aware)
- Tap targets >= 48px, ideally 52px in sticky rail
- Max 75 chars per line body text on phones
- Horizontal service card snap with momentum scrolling
- Keyboard-safe form spacing and submit visibility

## 5) Performance / Accessibility Targets
- Lighthouse mobile performance >= 90
- LCP < 2.1s, CLS < 0.06, INP < 170ms
- Initial critical CSS <= 65KB
- Hero media <= 170KB webp/avif equivalent
- AA contrast compliance for body and CTA text
- Reduced-motion support enabled

## 6) Niche Override Notes
- Landscaping: green + earth, transformation-focused imagery
- Roofing: amber/orange urgency + storm-response trust
- HVAC: cool blue precision + comfort assurance
- Pool: aqua clarity + maintenance consistency
- Pest: lime slate + family-safe treatment confidence
- Handyman: warm amber + fix-right-first-time language
