# Niche Playbooks (Landscaping / Roofing / HVAC)

## 1) Landscaping

### Positioning Angles
- Curb appeal + property value growth
- Low-maintenance outdoor living
- Seasonal reliability and cleanup speed

### High-Converting Service Buckets
- Design + install
- Lawn care maintenance plans
- Irrigation/drainage fixes
- Hardscaping (patios/retaining walls)

### Offer Templates
- Free yard assessment + 12-month plan
- Spring/fall cleanup bundle discount
- New-client maintenance package

### Proof to Feature
- Before/after transformations
- Plant survival warranty details
- Maintenance route coverage map

## 2) Roofing

### Positioning Angles
- Storm damage response speed
- Insurance-claim support
- Warranty-backed workmanship

### High-Converting Service Buckets
- Roof replacement
- Leak diagnostics + emergency repair
- Inspection/certification
- Gutter + ventilation upgrades

### Offer Templates
- Free 21-point roof inspection
- Storm season priority service slot
- Financing options callout

### Proof to Feature
- License + insurance badges
- Material partner certifications
- Claim assistance success stories

## 3) HVAC

### Positioning Angles
- Fast comfort restoration
- Energy savings + lower bills
- Year-round reliability via maintenance plans

### High-Converting Service Buckets
- AC repair/replace
- Furnace/heat pump install
- Indoor air quality
- Preventive maintenance memberships

### Offer Templates
- Same-day diagnostic promise
- Tune-up + filter package
- Membership with priority dispatch

### Proof to Feature
- Technician certifications (NATE, etc.)
- Response time stats
- Utility rebate guidance

## Cross-Niche Messaging Guardrails
- Always include geography in hero/subhead
- Avoid generic claims ("best quality") without proof
- Put one primary conversion action per viewport
- Place phone number in header + sticky mobile CTA
