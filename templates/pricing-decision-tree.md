# Pricing Decision Tree — How to Quote Clients

## Step 1: Understand Scope
Ask:
- Timeline requirement (ASAP vs. flexible?)
- Complexity (simple vs. complex?)
- Support needed (one-time vs. ongoing?)
- Budget range (do they have one?)

## Step 2: Estimate Hours
Use these benchmarks:

| Service | Simple | Medium | Complex |
|---------|--------|--------|---------|
| Excel Formula Audit | 2 hrs | 4 hrs | 8 hrs |
| VBA Macro | 4 hrs | 8 hrs | 16 hrs |
| Power Query Pipeline | 3 hrs | 6 hrs | 12 hrs |
| Python Script | 2 hrs | 6 hrs | 16 hrs |
| Web Scraper | 4 hrs | 10 hrs | 20 hrs |
| Simple Web App | 20 hrs | 40 hrs | 80 hrs |
| API Integration | 5 hrs | 12 hrs | 30 hrs |
| Data Cleaning | 2 hrs | 6 hrs | 16 hrs |

## Step 3: Calculate Rate
Use $75–150/hr as baseline depending on complexity:
- $75/hr = Straightforward, low-skill work
- $100/hr = Standard freelance rate
- $125/hr = Premium (complex, high-value)
- $150/hr = Expert-level (rare, high-risk work)

## Step 4: Quote with 3 Tiers
ALWAYS offer Good/Better/Best:

**Example: Excel Audit**
- Good ($150): Formula audit + report
- Better ($300): Audit + optimization + basic macro
- Best ($500): Full audit + optimization + advanced automation + training

Client picks the tier that fits their budget.

## Step 5: Add Buffer
Multiply hours × rate × 1.15 (15% buffer for unknowns).

## Step 6: Don't Negotiate Down
- If they want lower price, offer reduced scope instead
- "I can do X for $300, or a simpler version for $200"
- Never cut rate — that devalues your work

## Examples

### Excel Formula Audit (Simple)
- 2 hrs × $100 = $200 → Quote $250
- Scope: Review formulas, identify errors, optimization suggestions

### VBA Macro (Medium)
- 8 hrs × $100 = $800 → Quote $900
- Scope: Build macro, test, document

### Python Data Pipeline (Complex)
- 16 hrs × $125 = $2000 → Quote $2300
- Scope: Design pipeline, build & test, deploy, train client

### Custom Web App (Medium)
- 40 hrs × $100 = $4000 → Quote $4500 (or 3 tiers)
- Good: MVP only ($2000)
- Better: MVP + deployment ($4500)
- Best: Full app + hosting + support ($6000)

## Red Flags (Quote Higher or Decline)
- Vague scope ("make it work")
- Unrealistic timeline (30-hour project in 2 days)
- No budget mentioned
- Multiple revisions expected
- Client says "I'll provide feedback as we go" (scope creep risk)

→ Add 20% if any red flags present.
→ Decline if client seems difficult in initial conversation.
