## 4) Simple Demo Flow Script (3–5 Minutes)

### Demo Goal
Show how NorthStar turns one messy manual workflow into a reliable automation with visible business impact.

### Demo Setup (Before call)
- Prepare a sample workflow: intake form → spreadsheet/CRM → task + notification.
- Have “before” and “after” screens ready.
- Keep one real KPI visible (time saved, error drop, response speed).

### Talk Track + Sequence

#### Step 1 — Context (30 sec)
“Most teams we work with are doing critical ops via copy/paste, inbox monitoring, and spreadsheet cleanup. It works—until volume increases.”

#### Step 2 — Before State (45 sec)
Show the manual process:
- New request comes in
- Team member re-enters data manually
- Another person is pinged for follow-up
- Reporting gets updated later (or forgotten)

Say: “This is where delays and silent errors stack up.”

#### Step 3 — Automation Trigger (45 sec)
Show trigger event (new form submission/email/row):
- Data is captured once
- Validation runs automatically
- Record is created/updated in target system

Say: “No duplicate entry, no missed handoff.”

#### Step 4 — Routing + Alerts (45 sec)
Show assignment + notification logic:
- Right owner is assigned automatically
- Slack/email/task alert sent instantly
- SLA timer or reminder is started

Say: “Now ownership is automatic, and follow-through is visible.”

#### Step 5 — Reporting Snapshot (45 sec)
Show dashboard/report update:
- Status updated in real time
- Throughput and response metrics visible
- Exceptions flagged for quick intervention

Say: “Leadership gets live visibility without asking the team for updates.”

#### Step 6 — Business Impact Close (30 sec)
“Same process, less manual effort, faster response times, and fewer avoidable mistakes. This is the kind of workflow we can implement quickly in your environment.”

### Demo CTA Options
- **Primary:** “Let’s run a free automation audit on one workflow you want fixed this month.”
- **Secondary:** “We can scope a pilot automation and deliver first value in days, not months.”
