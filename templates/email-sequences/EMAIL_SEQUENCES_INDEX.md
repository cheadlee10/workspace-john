## Sequence Summary

### Sequence 1: No Website
- **Target:** Local businesses with zero web presence
- **Hook:** "Your {{X}} reviews are great but no one can find your website"
- **Proof:** Preview site already built for them
- **CTA:** "Worth a quick look?"
- **Best for:** Landscapers, contractors, home services

### Sequence 2: Outdated Website
- **Target:** Businesses with slow/ugly/poorly-ranking sites
- **Hook:** "I noticed {{specific issue}} with your site"
- **Proof:** Before/after comparison + case study
- **CTA:** "Worth a quick look?"
- **Best for:** Any business with identifiable web issues

### Sequence 3: Business Automation
- **Target:** Reddit/Upwork/Fiverr posters needing Excel/Python/bot work
- **Hook:** "Saw your post about {{their problem}}"
- **Proof:** Similar project experience
- **CTA:** "Want me to send a quick outline?"
- **Best for:** Inbound signals, technical buyers

## Universal Email Rules (All Sequences)

### Non-Negotiables
1. **Plain text only** — No HTML, no formatting, no images
2. **Under 100 words** — Body only, excluding signature
3. **No pricing in Email #1** — Save for follow-up after engagement
4. **Visual proof required** — Link to preview, screenshot, or portfolio
5. **Soft CTA only** — Question format, low friction
6. **Encoding clean** — No "?" characters, no garbled text
7. **3-7-7 cadence** — Day 0, Day 3, Day 10, (optional Day 17 breakup)

### Pre-Send Checklist (Every Email)
- [ ] First name verified and spelled correctly
- [ ] Company name matches their branding exactly
- [ ] Specific observation is TRUE and verifiable
- [ ] All links tested and working
- [ ] Pasted through plain-text editor
- [ ] Follow-up sequence staged
- [ ] No pricing mentioned

## Subject Line Quick Reference

### Highest Converting Formats (By Data)
| Format | Open Rate | Example |
|--------|-----------|---------|
| Question with company name | 46% | `Quick question about {{CompanyName}}` |
| Observation hook | 40%+ | `Noticed something about {{CompanyName}}` |
| Simple idea | 38% | `Idea for {{CompanyName}}` |

### Never Use
- All caps words
- Emojis
- "FREE" or "URGENT"
- Pricing in subject
- Generic greetings ("Hello friend")

## CTA Library (Copy-Paste Ready)

### Lowest Friction (Default)
```
Worth a quick look?
```

### Slightly More Commitment
```

```

### Video Offer
```
Mind if I send a quick video walkthrough?
```

### Call Soft-Ask
```

```

### Reply Trigger
```

```

```

```

### Breakup CTAs
```

Just reply to this email anytime if things change.
```

## Timing Guidelines

### Best Send Times
- **Primary:** Tuesday/Thursday, 9:00-11:00 AM local time
- **Secondary:** Wednesday, 1:00-3:00 PM local time
- **Acceptable:** Mon-Fri, 8:00 AM - 5:00 PM local time
- **Avoid:** Weekends, before 8 AM, after 6 PM

### Follow-Up Cadence
- **Email 1 (Day 0):** Initial outreach
- **Email 2 (Day 3):** Gentle bump + new insight
- **Email 3 (Day 10):** Case study / new value
- **Email 4 (Day 17):** Breakup email (optional but high-converting)

### Response Time
- If prospect replies: **Respond within 2 hours**
- 75% of email opens happen within 1 hour of sending

## Volume Operations (5K/week)

### Daily Targets
- 714 emails/day (7 days)
- 1,000 emails/day (5 days)

### Quality Control at Scale
- **Full QA:** First 5 emails per template variant
- **Spot check:** Random 10% of batch (minimum 20)
- **Automated:** 100% scanned for encoding, placeholders, links

### Batch Workflow
1. Research 50-100 prospects
2. Categorize by sequence (1, 2, or 3)
3. Fill personalization variables
4. Run encoding wash (plain-text paste)
5. Run automated QA scan
6. Spot-check 10%
7. Schedule sends (optimal timing)
8. Stage follow-up sequences
