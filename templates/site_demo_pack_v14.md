## 8) Simple Demo Flow Script (3–4 Minutes)

### Demo Objective
Show a clear before/after for a common operations flow:  
**Lead Intake → Assignment → Follow-Up → Reporting**

### Demo Setup (before the call)
- One sample web form (or intake form)
- Mock records in CRM/sheet
- Prebuilt automation scenario connected and ready
- Dashboard with fields: owner, status, SLA timer, follow-up state

### Live Script

**Step 1 — Frame the manual pain (30 sec)**  
“Today, this process depends on someone manually checking entries, assigning owners, sending follow-up, and updating status. That’s where delays and dropped tasks happen.”

**Step 2 — Trigger a new record (20 sec)**  
“Let’s submit one lead live.”

**Step 3 — Show automation in motion (60–75 sec)**  
“As soon as it lands, the workflow tags priority, assigns ownership, sends follow-up, and updates the dashboard automatically.”

**Step 4 — Show guardrails (45 sec)**  
“If data is incomplete or urgency is high, the workflow escalates and alerts the right person immediately.”

**Step 5 — Show measurable outcome (45 sec)**  
“Now we’ve reduced manual touches, improved response time, and created real-time visibility without extra admin work.”

### Close (15 sec)
“If this mirrors your workflow, we can map your exact process in a short audit and prioritize the quickest ROI wins.”
