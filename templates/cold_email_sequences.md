# QUALITY CHECKLIST (Run Before Every Send)

## Personalization
- [ ] First name spelled correctly
- [ ] Company name matches their branding
- [ ] City/location accurate
- [ ] Observation is true and verifiable
- [ ] Social proof example is real

## Content
- [ ] Under 100 words
- [ ] No pricing in Email #1
- [ ] No feature lists
- [ ] Soft CTA (question, not demand)
- [ ] Visual/preview link included and tested

## Technical
- [ ] No encoding errors (paste through Notepad first)
- [ ] All links working
- [ ] Professional sender domain
- [ ] Signature with phone number

## Compliance
- [ ] Physical address available
- [ ] Opt-out honored
- [ ] No deceptive subject lines

# USAGE NOTES

1. **Run QA Linter:** `python scripts/qa_email_lint.py email.txt` before sending
2. **Best Send Times:** Tuesday-Thursday, 9-11am or 1-2pm local time
3. **Visual Proof Required:** Always include preview link for web design; proposal link for automation
4. **Stop Sequence on Reply:** If prospect responds, halt automation immediately
5. **Log Everything:** Track opens, replies, conversions in leads.jsonl
